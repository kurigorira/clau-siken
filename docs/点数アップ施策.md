# 才弥さん 点数アップ施策（Ⅰ学期期末テスト）

テスト日程：社会6/29 → 数学7/3 → 国語7/6 → 英語7/10 → 理科7/13（すべて4校時）

---

## 1. 全体方針（いちばん効く3つ）

1. **「ワーク2周＋間違い直し」を最優先**
   どの教科も先生のアドバイスは「ワークを繰り返す」。1周目で印をつけ、2周目は×だけ、3周目は本番前日。これが最も点に直結する。
2. **提出物を“締切前”に終わらせる**
   提出物は内申にも関わる。社会6/29・理科7/1・英語7/10。特に理科は「丸付け・訂正まで」なので早めに。`go run . schedule` のリマインドで管理。
3. **間違えた問題＝伸びしろ。捨てずに記録する**
   「間違いノート」を1冊。間違えた問題と正解だけを書き、テスト前日に見返す。新しい問題を解くより効果的。

## 2. 教科別の作戦

### 社会（6/29）暗記＋流れ
- 年号の丸暗記より**「なぜ→どうなった」の流れ**で覚える（例：世界恐慌→各国のブロック経済→対立）。
- 教科書の**地図・資料・グラフ**は出題されやすい。語句と一緒に確認。
- **時事問題**は教科書に載っていない。4〜6月のニュースを親子で5分振り返る／新聞の見出しをメモ。
- 単元テストの解き直しは「先生が重視する論点」のヒント。

### 数学（7/3）反復で計算を速く正確に
- 展開・因数分解・根号は**スピードと正確さ**が命。毎日10問の計算ドリルでミスを減らす。
- 符号ミス・√の計算ミスが失点の大半。**途中式を必ず書く**。
- 「証明」など応用は配点が高い。**型（仮定→根拠→結論）を覚える**。
- `go run . quiz 数学` で毎日タイムを意識して挑戦。

### 国語（7/6）漢字で確実に得点
- **漢字は満点を狙える**。範囲の漢字を読み書き両方で。小テストの再確認を。
- 『握手』など本文は**授業プリントの読み直し**が近道（先生の解釈＝答え）。
- 俳句は**季語・五七五・表現技法（切れ字など）**を確認。
- ワークP6〜47の提出を忘れずに。

### 英語（7/10）現在完了が山場
- 3年最初の山＝**現在完了（経験・継続・完了）**。have/has＋過去分詞を体に入れる。
- 不規則動詞の**過去分詞**を毎日5個ずつ。
- 教科書の**対話文（Aさん・Bさんの文）**を音読して暗唱。本文がそのまま問題になりやすい。
- ワークは2周（P35を除く範囲）。提出は7/10。

### 理科（7/13）計算＋作図は必出
- 先生明言「**作図・計算問題は必ず出る**」。仕事＝力×距離、速さ＝距離÷時間、仕事率＝仕事÷時間の3公式を確実に。
- 力の合成・分解、斜面・自由落下、力学的エネルギー保存をノートで図解。
- 単位（N・J・W）をセットで覚える。
- ワークは7/1提出（丸付け・訂正まで）→ 提出後も2周目で仕上げ。

## 3. 1日の進め方（30分〜2時間）

- **平日60分／土日120分**を目安（`schedule` コマンドが自動配分）。
- 区切りは**25分集中＋5分休憩**（ポモドーロ）。スマホは別室。
- 順番は「①前日の間違い直し5分 → ②今日のワーク → ③`quiz` で確認」。
- 寝る前に**今日覚えたことを1分で口に出す**（記憶の定着）。

## 4. 本番で点を落とさない

- 名前・記号・単位の書き忘れチェック。
- 分かる問題から解き、難問は飛ばして後で戻る。
- 見直し5分を必ず残す（特に数学の符号、理科の単位）。

## 5. 保護者のサポート

- 結果ではなく**取り組み（机に向かった事実）をほめる**。
- 提出物の締切だけ一緒に確認（社会6/29・理科7/1・英語7/10）。
- 時事問題のニュースを夕食時に雑談で。
