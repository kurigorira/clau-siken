# clau-siken — 才弥さん 期末テスト対策ツール

令和8年度 第3学年 Ⅰ学期期末テストの **学習スケジュール** と **問題集（クイズ）** を
扱う Go 製コマンドラインツールです。点数アップの作戦は `docs/点数アップ施策.md` にまとめています。

## テスト日程

| 回 | 日付 | 教科 |
|----|------|------|
| ① | 6/29(月) | 社会 |
| ② | 7/3(金) | 数学 |
| ③ | 7/6(月) | 国語 |
| ④ | 7/10(金) | 英語 |
| ⑤ | 7/13(月) | 理科 |

※すべて4校時に実施

## 使い方

Go 1.21+ が必要です。**引数なしで起動すると、番号を選ぶだけの対話メニュー**になります。

```sh
go run .
```

```
==================================================
 才弥さん 期末テスト対策ツール
 今日 6/13(土) ／ 次のテスト：社会（あと16日）
==================================================
  1) 今日やること（スケジュール）
  2) 問題集に挑戦
  3) テスト範囲・提出物を見る
  4) 点数アップのヒント
  0) おわる
えらんでね >
```

番号を入れて Enter で進み、操作のあとはメニューに戻ります。`2`（問題集）を選ぶと
さらに教科を番号で選べます。

コマンドを直接指定することもできます（上級者向け）：

```sh
go run . schedule    # 学習スケジュール全日程を表示
go run . scope       # 各教科のテスト範囲・提出物・先生の助言を表示
go run . quiz        # 全教科の問題集に挑戦
go run . quiz 数学    # 教科を指定して挑戦（社会 / 数学 / 国語 / 英語 / 理科）
```

ビルドして実行ファイルにする場合：

```sh
go build -o saiya-study .
./saiya-study          # メニューが起動
```

## 構成

| ファイル | 役割 |
|----------|------|
| `main.go` | 起動処理・コマンド振り分け |
| `menu.go` | 番号で選ぶ対話メニュー |
| `data.go` | 教科・テスト範囲・提出物・問題データ |
| `schedule.go` | 日別スケジュールの自動生成と表示 |
| `quiz.go` | 対話形式の問題集と答え合わせ |
| `docs/点数アップ施策.md` | 教科別の勉強法・点数アップの作戦 |

## ポイント

- スケジュールは **平日60分・土日120分** を基本に、直近のテスト教科へ自動配分。
  テスト2日前は「総仕上げ」に切り替わります。
- 提出物（社会6/29・理科7/1・英語7/10）は締切3日前からリマインド表示。
- クイズの答え合わせは全角／半角・空白のゆれを吸収します（例：`２√３` も `2√3` も正解）。
